## Usage

```tsx
import { StampImage } from "@/components/stamp";

// Basic usage - simple stamp conversion
<StampImage src="/path/to/image.jpg" />

// Custom size stamp
<StampImage 
  src="/path/to/image.jpg" 
  size={200}
  innerColor="#ffffff"
  outerColor="#c0c0c0"
/>

// Vintage stamp with custom colors
<StampImage 
  src="/path/to/image.jpg" 
  size={150}
  innerColor="#f8f8f8"
  outerColor="#8b4513"
  className="shadow-lg"
/>

// Large stamp for display
<StampImage 
  src="/path/to/image.jpg" 
  size={300}
  innerColor="#fffef7"
  outerColor="#d4af37"
  onLoad={() => console.log('Stamp loaded')}
  onError={() => console.log('Error loading image')}
/>
```

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `src` | `string` | ✅ | - | The source path or URL of the image to convert to a stamp. |
| `alt` | `string` | ❌ | `"Stamp image"` | Alternative text for accessibility. |
| `size` | `number` | ❌ | `200` | Size of the stamp in pixels. If not provided, uses original image dimensions. |
| `className` | `string` | ❌ | `""` | Additional CSS classes to apply to the stamp canvas. |
| `onLoad` | `() => void` | ❌ | - | Callback fired when the image loads successfully. |
| `onError` | `() => void` | ❌ | - | Callback fired when the image fails to load. |
| `innerColor` | `string` | ❌ | `"#ffffff"` | Color for the inner part of the stamp (background). |
| `outerColor` | `string` | ❌ | `"#c0c0c0"` | Color for the outer part of the stamp (border, perforation). |

## Features

- **Perforated Borders**: Realistic stamp perforation effects
- **Customizable Colors**: Choose inner and outer colors
- **Responsive Sizing**: Automatic aspect ratio preservation
- **Error Handling**: Graceful fallback for failed image loads
- **Canvas Rendering**: High-quality image processing
- **Accessibility**: Proper ARIA labels and alt text
- **Drop Shadow**: Built-in shadow effects for realism
