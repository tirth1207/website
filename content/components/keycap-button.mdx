## Usage

```tsx
import KeycapButton from "@/components/KeyCapButton";

// Basic usage - simple keycap button
<KeycapButton>Enter</KeycapButton>

// Common keyboard keys
<KeycapButton>Ctrl</KeycapButton>
<KeycapButton>Alt</KeycapButton>
<KeycapButton>Space</KeycapButton>
<KeycapButton>Shift</KeycapButton>

// Special characters
<KeycapButton>⌘</KeycapButton>
<KeycapButton>⌥</KeycapButton>
<KeycapButton>⌃</KeycapButton>

// Function keys
<KeycapButton>F1</KeycapButton>
<KeycapButton>F12</KeycapButton>
<KeycapButton>Esc</KeycapButton>
```

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `children` | `string` | ✅ | - | The text content to display on the keycap button. |
